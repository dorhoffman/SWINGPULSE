# 🚀 SWINGPULSE

An AI-powered stock market analysis platform built as a B.Sc. Information Systems final project.

## Project Overview

SWINGPULSE combines:

- 📈 Technical Analysis
- 🤖 Machine Learning
- 🧠 AI Agent
- 📊 Data Analytics
- 🔍 Similarity Search
- 🧩 Clustering

to support swing trading decisions using historical stock market data.

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Google Colab
- GitHub
- Yahoo Finance Dataset
