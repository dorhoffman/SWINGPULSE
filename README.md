 📈 SwingPulse AI

An AI-powered swing trading assistant that combines machine learning, technical analysis, and an LLM to help users analyze stocks using natural language.

Supports both **English** and **Hebrew**.

## Features

- 📊 Analyze any stock
- 📈 Technical indicators (RSI, MACD, EMA, SMA, Bollinger Bands, ATR)
- 🤖 ML probability predictions
- 🔍 Compare multiple stocks
- 💬 Natural language chat interface
- 🌐 Live market data

## Tech Stack

- Python
- Streamlit
- OpenAI API
- Scikit-learn
- yfinance
- Pandas
